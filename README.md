# Deep Researcher On Dify — README

**Deep Researcher On Dify** 是一个基于 Dify 打造的多阶段深度研究助手，通过精细化的 Prompt 设计，实现对用户研究主题的澄清、修正、扩展、结构化与报告生成的全流程自动化。它适用于从初步概念到完整学术或专业报告的系统化研究任务。

## 🔍 功能概述

本研究助手通过多步骤协作流程，实现：

### 1. 研究主题澄清与方向聚焦
系统会自动对用户初始研究主题提出 1–4 个澄清性问题，以避免主题偏移、指向不清或研究范围过大。

### 2. 主题修正与专业化处理
结合用户的问答内容，对研究主题进行局部至整体的增强，包括：
- 逻辑结构强化  
- 专业深度提升  
- 背景和细节点补全  
- 语义优化与研究价值提升  

### 3. 针对用户问答的深度扩展
对用户提供的 Q&A（q1/f1、q2/f2 等）进行进一步的推理与内容扩展，使回答更：
- 具深度  
- 连贯  
- 有学术性和说服力  
- 贴合专业报告标准  

### 4. 自动生成结构化子标题
根据原始主题与修订主题，生成 3–5 个逻辑递进、不重叠的报告小标题（subtitles），确保主题展开结构清晰。

### 5. 生成完整报告的开头与结尾段落
助手可自动生成：
- **专业、自然过渡的报告开篇段落**（引入背景并承接第一小标题）  
- **严谨的学术式总结段落**（在正文后输出）  

确保整份报告在学术逻辑、文风与结构上保持一致。

## 🧠 适用场景

- 学术研究主题构思与深化  
- 商业或技术类深度调研  
- 报告/白皮书自动化生成  
- 市场、政策、行业分析  
- 多步骤分析类任务（需要从主题 → 结构 → 内容 → 输出）

## 🛠️ 优势

- 全流程自动化，无需手动引导模型  
- 精准避免主题偏移  
- 保持高水准、强逻辑的学术写作风格  
- 可持续迭代优化用户提供的材料  
- 支持多语言输出  

## 📦 集成方式

直接将本项目作为 Dify 应用或工作流导入，输入研究主题以及相关问答内容，即可开始深度研究流程。
